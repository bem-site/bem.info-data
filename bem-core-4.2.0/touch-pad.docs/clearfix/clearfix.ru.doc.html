clearfix

Блок предоставляет СSS-класс, реализующий прием верстки clearfix, также известный как Easy Clearing Hack. Прием позволяет отменить обтекание для элементов с CSS-свойством float, без внесения изменений в исходную HTML-структуру документа.

Блок можно использовать в качестве контейнера для элементов со свойством float, или примешивая его к такому контейнеру.
